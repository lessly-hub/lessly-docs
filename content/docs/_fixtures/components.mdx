## Callout — three variants

<Callout variant="note">
Use the `lessly_list_deployments` tool to enumerate every deployment in the active workspace. Returns up to 50 entries per call.
</Callout>

<Callout variant="warning" title="Heads up">
The MCP server caches workspace tokens for 5 minutes. After rotating a token you may see a stale auth error for one cycle.
</Callout>

<Callout variant="danger">
`lessly_delete_deployment` is irreversible. There is no undo and no grace period — the worker is torn down immediately.
</Callout>

<Callout variant="info">
This should render as a `note` and emit a build-time console warning.
</Callout>

## Tabs — install variants

<Tabs id="install" defaultTab="claude-code">
  <Tab value="claude-code" label="Claude Code">
    Run this in your terminal:

    ```bash
    claude mcp add lessly https://mcp.lessly.com
    ```
  </Tab>
  <Tab value="claude-desktop" label="Claude Desktop">
    Edit `~/Library/Application Support/Claude/claude_desktop_config.json`:

    ```json
    {
      "mcpServers": {
        "lessly": { "url": "https://mcp.lessly.com" }
      }
    }
    ```
  </Tab>
  <Tab value="cursor" label="Cursor">
    Settings → MCP → Add Server → `https://mcp.lessly.com`.
  </Tab>
</Tabs>
