# Get started

## Goal

By the end of this page, your AI agent can call Lessly tools to deploy and manage your apps.

## Prerequisites

- A Lessly workspace — [sign up at lessly.com](https://lessly.com/signup) (Google sign-in).
- A supported AI agent (see tabs below).

## Steps

1. Open your Lessly workspace and find the **Connections** panel.
2. Copy the Lessly MCP URL.
3. Add it to your agent following the tab below.
4. Authenticate via the browser when prompted.
5. Try a first command: *"Show me my recent deployments."*

### Add Lessly to your agent

<Tabs items={['Claude Desktop', 'Claude Code', 'Cursor']}>
  <Tab value="Claude Desktop">
    Open `~/Library/Application Support/Claude/claude_desktop_config.json` and add:

    <CodeBlock language="json" filename="claude_desktop_config.json">
{`{
  "mcpServers": {
    "lessly": {
      "url": "https://mcp.lessly.com"
    }
  }
}`}
    </CodeBlock>

    Restart Claude Desktop. Sign in when prompted.
  </Tab>

  <Tab value="Claude Code">
    <CodeBlock language="bash">
{`claude mcp add lessly https://mcp.lessly.com`}
    </CodeBlock>

    Sign in when prompted in the browser.
  </Tab>

  <Tab value="Cursor">
    In Cursor, open **Settings → MCP → Add Server**. Paste `https://mcp.lessly.com`. Authenticate in the browser when prompted.
  </Tab>
</Tabs>

## You just did this

Your agent now has the `lessly_*` toolset. Try asking:

> Deploy my latest commit to staging.
