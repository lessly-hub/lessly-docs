By the end of this page, your AI agent can call Lessly tools to deploy and manage your apps. Installation takes about 30 seconds in Claude Code.

## Prerequisites

- A Lessly workspace — [sign up at lessly.com](https://lessly.com/signup) (Google sign-in).
- A supported AI agent: Claude Code, Claude Desktop, or Cursor.

## Install in Claude Code

The fastest path. One command in your terminal:

```bash
claude mcp add lessly https://mcp.lessly.com
```

Claude Code opens your browser to authenticate. Sign in with the same Google account you used for Lessly. Once the browser confirms, the connection is live in Claude Code — no restart needed.

## Install in Claude Desktop

Open the config file at `~/Library/Application Support/Claude/claude_desktop_config.json` and add the Lessly server:

```json
{
  "mcpServers": {
    "lessly": {
      "url": "https://mcp.lessly.com"
    }
  }
}
```

Quit and reopen Claude Desktop. Sign in when prompted.

## Install in Cursor

In Cursor, open **Settings → MCP → Add Server**. Paste `https://mcp.lessly.com` as the server URL. Authenticate in the browser when prompted.

## Verify the install

Ask your agent:

> Show me my recent deployments.

The agent will call the `lessly_list_deployments` tool. If you see a list (or an empty list with a confirmation that the tool worked), the install is complete.

## What you can do next

Your agent now has the full `lessly_*` toolset. Try:

- *"Deploy my latest commit to staging."*
- *"Roll back the last production deploy."*
- *"What environment variables are set on the api service?"*

If a command fails, check that you signed in with the correct workspace — the MCP shows the active workspace name in its initialization message.
